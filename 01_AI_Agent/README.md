# 01 — AI Chat Agent

This folder contains all configuration, flows, and prompt files for the AI qualification agent.

## Contents
- agent_flow.json — Exported flow from Voiceflow / Chatbase
- qualifying_questions.md — Questions the agent asks
- lead_scoring_logic.md — How Hot / Warm / Cold scores are calculated
- system_prompt.md — System prompt for the AI agent

## Agent Behaviour
1. Greets the prospect warmly
2. Collects: Name, Email, Phone, Company, Services, Budget, Timeline
3. Scores the lead: Hot / Warm / Cold
4. Hot Lead → Triggers calendar booking
5. Warm/Cold Lead → Adds to nurture list
